1. Назначение проекта
Разработка веб-сайта для обмена книгами с интеграцией ИИ (ChatGPT 4.0) для помощи пользователям в выборе книг. Сайт предназначен для упрощения процесса поиска, обмена и рекомендации книг среди пользователей.

2. Цели и задачи
Цель:
Создать онлайн пространство(сайт), в котором можно найти подходящую для себя книгу, обменять свою на чужую и получить рекомендации от Нейросети с учетом личных пожеланий.
Задачи:
Реализовать backend на Python с использованием Flask.
Разработать frontend с использованием HTML, CSS и JavaScript.
Интегрировать ИИ (вероятно ChatGPT) для рекомендаций книг.
Обеспечить хранение данных в SQLite.
Разместить сайт на облачном сервисе (PythonAnywhere).

3. Функциональные требования
Основные модули сайта:
Главная страница
Краткое описание сервиса.
Навигация по разделам.
Регистрация и вход
Формы для регистрации и авторизации пользователей.
AI Book Assistant
Чат-бот на базе ChatGPT 4.0 для рекомендаций книг.
Галерея книг
Отображение доступных для обмена книг с фильтрами по жанрам, авторам и т.д.
Раздел "Поделиться книгой"
Форма для добавления книги в обмен.
Раздел "Найти книгу"
Поиск книг по названию, автору, жанру.
Раздел "О сайте"
Информация о проекте и его создателях, а также контактная информация.

4. Технические требования
Backend: Python с использованием бибилиотеки Flask.
Frontend: HTML, CSS, JavaScript.
База данных: SQLite.
Хостинг: PythonAnywhere.
Интеграция ИИ: API ChatGPT

Интуитивно понятный и удобный интерфейс.
Адаптивный дизайн для корректного отображения на разных устройствах.
